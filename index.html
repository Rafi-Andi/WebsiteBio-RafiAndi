ABOUT ME

Halo! Nama saya Moch Rafi Andi Prayitno, dan saya seorang siswa di SMKN 1 Surabaya di kelas 10 dengan jurusan Rekayasa Perangkat Lunak. Saya sangat antusias dalam belajar dan mengeksplorasi berbagai bidang, dan saya berusaha untuk memanfaatkan setiap kesempatan yang saya miliki untuk berkembang dan belajar lebih banyak.

Saat ini, saya terlibat dalam berbagai kegiatan ekstrakurikuler, termasuk Esktra Kulikuler ITC (IT Club) Divisi Programming aktivitas-aktivitas ini tidak hanya membantu saya mengembangkan keterampilan baru, tetapi juga memungkinkan saya untuk berkolaborasi dengan teman-teman dan berkontribusi kepada komunitas sekolah.

Di luar sekolah, Saya sangat menyukai Pemprograman Website sesuai dengan jurusan saya, dan saya berusaha untuk mendalami lebih jauh tentang hal tersebut.
